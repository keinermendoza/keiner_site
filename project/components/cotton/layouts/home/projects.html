Es un sitio web enfocado en la promoción de cursos publicados en la plataforma Hotmart.

Los usuarios pueden buscar y filtrar cursos, mientras que los administradores del sitio cuentan con un panel de control con un editor integrado para modificar la información de los cursos, incluyendo la gestión de imágenes.

Realicé el diseño utilizando Figma y lo implementé con Tailwind CSS. Desarrollé una API REST con Django, que incluyó el manejo de archivos e integré el editor WYSIWYG Editor.js con React. Finalmente, aprovisioné el servidor, configuré la base de datos y habilité un servicio de almacenamiento para gestionar los archivos.
